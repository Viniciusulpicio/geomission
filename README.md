# GeoMission: Módulo de Geometria Plana

### Descrição do Projeto

GeoMission é uma aplicação web interativa e gamificada desenvolvida para auxiliar no ensino e aprendizagem de Geometria Plana no Ensino Médio.

O projeto utiliza uma interface moderna com temática "Sci-Fi" (Ficção Científica) para engajar os alunos, transformando exercícios matemáticos abstratos em missões visuais. A aplicação roda inteiramente no navegador, sem necessidade de instalação de softwares pesados ou conexão constante com a internet, facilitando seu uso em laboratórios de informática ou dispositivos móveis.

Este software foi desenvolvido como parte de uma atividade avaliativa da disciplina de Matemática no SENAI Shunji Nishimura.

### Funcionalidades

O sistema possui dois modos de operação distintos:

1. Modo Estudo (Solo)

Focado na prática individual e fixação de conteúdo. O aluno pode escolher entre três temas principais:

* Perímetros: Cálculo da soma dos lados de figuras geométricas (Retângulos).

* Áreas: Cálculo da superfície interna de figuras (Retângulos e Triângulos).

* Teorema de Pitágoras: Cálculo da hipotenusa em triângulos retângulos.

* Recurso de Ajuda: Um botão que revela a fórmula matemática necessária caso o aluno tenha dúvidas.

2. Duelo Geométrico (Multiplayer Local)

Focado na competição e agilidade de raciocínio.

* Tela Dividida (Split-Screen): Dois jogadores utilizam o mesmo dispositivo (PC ou Tablet).

* Desafios Aleatórios: O sistema gera automaticamente problemas de área, perímetro ou Pitágoras.

* Dinâmica: Vence quem calcular e inserir a resposta correta primeiro.

### Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web padrão, garantindo compatibilidade com qualquer navegador moderno (Chrome, Edge, Firefox, Safari):

* HTML5: Estrutura semântica da aplicação.

* CSS3 & Tailwind CSS: Estilização responsiva e design visual (via CDN).

* JavaScript (ES6+): Lógica de programação, geração de problemas matemáticos aleatórios e manipulação do DOM.

* HTML5 Canvas API: Renderização gráfica dinâmica das figuras geométricas (desenho de triângulos e retângulos em tempo real).
